# 制作背景
本アプリは、動物病院向けのシンプルな電子カルテシステムです。想定ユーザーは、動物病院で働く獣医師や動物看護師です。
エンジニアへの転職を目指す中で、自分の強みである「動物医療の現場経験」と、新たに身につけた「プログラミングスキル」を掛け合わせたポートフォリオを作成したいと考え、本アプリを開発しました。
